# JUUNÉ Skin Atelier

A premium, luxury skincare clinic landing page built with Next.js 15, React 19, and pure, highly optimized CSS.

## Features
- **Luxury Editorial Aesthetic**: Custom typography scaling, smooth scroll-reveal animations, and a calm, sophisticated color palette.
- **Fully Responsive Architecture**: Fluid typography and spacing using `clamp()` calculations for seamless scaling across mobile, tablet, and desktop.
- **Component-Driven**: Modular design system using pure CSS for maximum performance and minimum bloat.
- **Optimized Assets**: Next.js optimized images (`next/image`) and fonts (`next/font`) for instantaneous load times.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- **Framework**: Next.js App Router
- **Styling**: Pure CSS (Design System Tokens)
- **Typography**: Cormorant Garamond (Headings) & Inter (Body)
